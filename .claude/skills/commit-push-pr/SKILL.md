# Commit-Push-PR Skill

変更内容のコミット、リモートへのプッシュ、Pull Request の作成・更新を一括で実行する。

## 手順

以下の 3 ステップを順番に実行する。各ステップは対応するスキルの手順に従う。

### ステップ 1: コミット（`/commit` スキル）

`.claude/skills/commit/SKILL.md` の手順に従い、Conventional Commits 規約でコミットを作成する。

- 未コミットの変更がない場合はこのステップをスキップし、ステップ 2 へ進む

### ステップ 2: プッシュ（`/push` スキル）

`.claude/skills/push/SKILL.md` の手順に従い、リモートへプッシュする。

- リモート未プッシュのコミットがない場合はこのステップをスキップし、ステップ 3 へ進む

### ステップ 3: PR 作成（`/pr` スキル）

`.claude/skills/pr/SKILL.md` の手順に従い、Pull Request を作成または更新する。

- `/pr` スキル内の「未コミット変更の警告」「未プッシュコミットの警告」はステップ 1・2 で処理済みのためスキップする

## 引数の受け渡し

- `--draft` が指定された場合は `/pr` スキルに渡す
- その他の引数は適切なスキルに渡す

## 注意事項

- 各ステップでエラーが発生した場合はそこで停止し、ユーザーに報告する。後続ステップは実行しない
- 各スキルの注意事項・制約をすべて遵守する
